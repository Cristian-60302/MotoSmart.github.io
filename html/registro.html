i Tratamiento de datos personales

MotoSmart es un software academico desarrollado por estudiantes. Los datos que ingreses se utilizan unicamente para demostrar el funcionamiento del registro, inicio de sesion y visualizacion de informacion de la motocicleta dentro del proyecto.

  • No se vendera ni compartira tu informacion con terceros.
  • El correo se usa para validar tu acceso a la plataforma.
  • Los datos de la moto se usan para personalizar la ficha tecnica y el dashboard.
  • Este proyecto tiene fines educativos y no reemplaza servicios tecnicos oficiales.
